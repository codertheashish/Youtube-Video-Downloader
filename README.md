# 🎬 YouTube Video Downloader

A modern YouTube Video Downloader built using **Python, Flask, HTML, CSS, JavaScript, and yt-dlp**. It allows users to download YouTube videos in **MP4** format or extract audio as **MP3** with a simple and responsive web interface.

---


## ✨ Features

- 📥 Download YouTube Videos (MP4)
- 🎵 Extract Audio as MP3
- 🔗 Paste Any YouTube Video URL
- ⚡ Fast Download Processing
- 🌐 Clean & Responsive User Interface
- 📱 Mobile-Friendly Design
- 🎨 Modern Web UI
- 🚀 Powered by yt-dlp
- 🔒 Simple & Easy to Use

# 🚀 Getting Started

### 1. Clone Repository

```bash https://github.com/codertheashish/Youtube-Video-Downloader.git
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install FFmpeg

Windows:
Download FFmpeg from:
https://ffmpeg.org/download.html

Mac:

```bash
brew install ffmpeg
```

Linux:

```bash
sudo apt install ffmpeg
```

### 4. Run Project

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

# ⚙️ How It Works

- Paste a YouTube video URL
- Select MP4 or MP3 format
- Process the video using yt-dlp
- Download the file directly to your device
